# Problem: Task Object Information

Create an object (JavaScript) or dictionary (Python) that represents a task.

The task should contain the following information:

- title
- completed
- priority

Example task:

title: "Study programming"  
completed: false  
priority: "high"

Your program should:

1. Create the task object/dictionary
2. Print the task title
3. Print whether the task is completed
4. Print the task priority

Example Output:

Task: Study programming  
Completed: false  
Priority: high
