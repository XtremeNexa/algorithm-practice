# Algorithm Thinking Practice

A collaborative repository where beginner developers practice core programming concepts together using different languages.

The goal is to focus on **problem-solving and algorithmic thinking**, not just language syntax.

Each problem is solved in multiple languages so contributors can compare logic across ecosystems such as JavaScript and Python.

---

## Goals

- Strengthen programming logic
- Practice algorithm-first thinking
- Learn language-independent problem solving
- Improve collaboration between developers using different languages

---

## Learning Topics

We practice fundamental programming concepts including:

- Loops
- Arrays / Lists
- Conditions
- Functions
- Objects / Dictionaries
- Searching algorithms
- Sorting algorithms
- String manipulation
- Basic data structures

---

## Repository Structure

Each problem follows the same structure:
| Path                        | Description              |
|-----------------------------|--------------------------|
| problem-name/               | Root folder              |
| ├─ problem.md               | Problem description      |
| ├─ algorithm.md             | Algorithm explanation    |
| ├─ solution-js.js           | JavaScript implementation|
| └─ solution-python.py       | Python implementation    |
---

### Files Explained

**problem.md**

Contains the problem description and example input/output.

**algorithm.md**

Step-by-step logic written in plain language (language independent).

**solution-js.js**

JavaScript implementation.

**solution-python.py**

Python implementation.

---

## Example

Problem: Sum all numbers in an array.

Algorithm:

1. Start with sum = 0
2. Loop through each number
3. Add the number to sum
4. Print the result

Each contributor then implements the solution in their own language.

---

## How to Contribute

1. Pick a problem
2. Write the algorithm
3. Implement the solution in your language
4. Submit a commit
